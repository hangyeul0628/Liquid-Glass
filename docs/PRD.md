# [PRD] '글래스모피즘 쉴드' 스크롤텔링 랜딩 페이지 제품 요구사항 정의서

본 문서는 리퀴드 글래스 페이드(Liquid Glass Fade) 모션 콘셉트와 맑은 유리알 컬러 팔레트를 기반으로 한 화장품 제품 홍보 랜딩 페이지의 프론트엔드 개발 및 UI/UX 설계를 위한 제품 요구사항 정의서(PRD)입니다.

## 1. 프로젝트 개요 및 기술 스택
### 1.1 프로젝트 기본 정보
* 프로젝트명: 글래스모피즘 쉴드 앰플 크림 랜딩 페이지 구축
* 목적: 타겟 페르소나(29세 직장인 김지현)의 시각적 정화와 소유욕을 자극하여 3일 체험 키트 신청 및 제품 구매(CTA) 유도
* 핵심 메시지: "푸석하게 깨진 장벽을 넘어, 당신의 피부에 완벽한 유리알 투명도를 박제하다."
* 페르소나가 느껴야 할 감정: 시각적 정화와 소유욕
### 1.2 기술 스택
* Framework / Build Tool: React (v18+) + Vite
* Styling: 일반 CSS (Vanilla CSS / CSS Modules)
* Motion / Interaction Libraries: Framer Motion, GSAP + ScrollTrigger

## 2. 디자인 시스템 정의
### 2.1 컬러 팔레트 (Clear Glass)
:root {
  --bg-color: #F4F7F6;
  --primary-color: #1E3A34;
  --accent-color: #A3C6C0;
  --point-color: #FF9E80;
  --white: #FFFFFF;
}
### 2.2 폰트 적용 가이드
* 강조용 (Heading): Pretendard-Bold 또는 Pretendard-ExtraBold
    * 적용 이유: 자로 잰 듯 반듯하고 현대적인 실루엣이 세련된 유리의 직선적 미학과 매칭되는 산세리프 폰트입니다. 밝은 배경 위에서도 명확한 시각적 대비와 단단한 신뢰감을 전달합니다.
* 본문용 (Body): RidiBatang
    * 적용 이유: 딱딱한 화면에 부드럽고 유기적인 호흡을 불어넣는 명조체입니다. 유연하고 편안한 리퀴드 감성을 담아 바쁜 직장인 타겟에게 나를 위한 휴식 같은 정서적 안도감을 줍니다.

## 3. 섹션별 상세 구성 및 모션 플로우
### SECTION 01: Hero Section
* 목적: 브랜드의 첫인상 각인 및 시각적 압도
* 레이아웃: 중앙 집중형 미니멀 레이어 폼 (display: flex; justify-content: center; align-items: center;)
* 포함 요소: 3D 유리 질감 제품 오브제, 메인 카피 타이포그래피, 스크롤 유도 다운 애로우 인디케이터
* 헤드라인 카피: "푸석한 거칠음을 지나, 내 피부에 완벽한 유리알 투명도를 박제하다."
* 모션 플로우:
    * 진입 시: 타이포그래피 블러(10px) 상태에서 선명해지며 페이드인. 중앙 3D 오브제 Y축 부유 모션. 커서 따라 은은한 광채 궤적 연출.
    * 스크롤 시: 3D 제품 오브제 줌인 확대.
    * 호버 시: 제품 오브제에 유리 반사 하이라이트 글레어 효과 마우스 위치 따라 이동.
### SECTION 02: The Break
* 목적: 피부 푸석함을 '장벽 균열'로 시각화하여 공감대 형성
* 레이아웃: 풀스크린 그리드 (height: 100vh; position: relative;)
* 포함 요소: 거칠고 탁한 유리 텍스처 배경, 균열 그래픽 소스, 경고성 서브 카피
* 헤드라인 카피: "밀리고, 뜨고, 빛을 잃은 얼굴. 당신의 수분 보호막에 미세한 균열이 시작됐기 때문입니다."
* 모션 플로우:
    * 진입 시: 01 섹션 줌인 화면이 거친 유리 텍스처로 자연스럽게 전환.
    * 스크롤 시: GSAP ScrollTrigger 연동, 스크롤 진척도에 따라 유리 표면에 균열(Cracked) 패스 드로잉. 배경 톤 탁해짐.
    * 호버 시: 균열 부위 커서 지나갈 시 미세 흔들림과 텍스처 반응.
### SECTION 03: The Rebuild
* 목적: '마이크로 젤-클레이' 제형의 복원력 시각화
* 레이아웃: 좌우 2단 분할 (좌: 텍스트 / 우: 인터랙티브 모션)
* 포함 요소: 3D 클레이 젤 파티클, 정화된 유리알 스킨 배경, 경쟁사 vs 자사 흡수 구조 비교 3D 단면 그래픽
* 헤드라인 카피: "틈새를 메우는 마이크로 젤-클레이 포뮬러. 무게감 없는 투명 유리막으로 코팅합니다."
* 모션 플로우:
    * 진입 시: 세이지 민트 컬러 쉴드가 위에서 아래로 물 흐르듯 덮는 Liquid Wipe Effect.
    * 스크롤 시: 균열 사이로 클레이 젤 파티클이 채워짐.
    * 호버/드래그 시: 마우스 드래그(Scratch & Reveal) 시 탁한 표면이 맑은 유리알 표면으로 치유되듯 복원.
### SECTION 04: The Proof
* 목적: 데이터 지수와 후기로 구매 의구심 제거
* 레이아웃: 고정 사이드바 인디케이터 + 카드 뉴스 롤링
* 포함 요소: 유리 실린더 동적 게이지 바, 임상 데이터 수치, 타겟 맞춤 후기
* 헤드라인 카피: "자극 없는 네오 내추럴 성분으로 이뤄낸 24시간 장벽 박제 효과. 24시간 흐려지지 않는 99% 투명도 지수."
* 모션 플로우:
    * 진입 시: 유리 실린더 UI 스티키 고정.
    * 스크롤 시: 리뷰 카드 스크롤에 맞춰 실린더 게이지 [35%] ➡️ [65%] ➡️ [99%] 실시간 상승, 코랄 핑크 컬러 채워짐.
    * 호버 시: 카드 호버 시 Y축 부상 및 은은한 반투명 드롭 섀도우 활성화.
### SECTION 05: CTA Section
* 목적: 최종 구매 유도
* 레이아웃: 풀화면 센터 플렉스, 은은한 그라데이션 배경
* 포함 요소: 리추얼 보증서 오버레이, 3가지 옵션 인터랙티브 버튼
* 헤드라인 카피: "오늘부터, 당신의 일상에 깨지지 않는 맑음을 선물하세요."
* 모션 플로우:
    * 진입 시: 몽환적 블러(20px) 상태에서 맑게 페이드인.
    * 스크롤 시: 세이지 민트에서 코랄 핑크로 부드러운 그라데이션 전환.
    * 호버 시: 버튼 내부 반투명 유리 레이어 이동 및 마우스 커서 끌어당김 인터랙션.

## 4. 인터랙션 포인트 마스터 목록
* 전역 커서 인터랙션: 물방울 파장(Ripple Effect) 및 광채 궤적.
* 스크롤 동기화 모션: 제품 줌인 및 유리 균열 드로잉 (GSAP ScrollTrigger).
* 글래스모피즘 스크래치: 캔버스 마스크를 이용한 불투명층 지우기 및 투명도 복원.
* 카운팅 애니메이션: 게이지 상승과 함께 투명도 수치 0~99% 카운팅.
* 블러 스위치: CSS filter: blur() 감소를 통한 시각적 정화 연출.

## 5. 모바일 대응 가이드
* 미디어 쿼리: Tablet(1024px), Mobile(768px).
* 규칙:
    * 커서 인터랙션: 터치 이벤트로 대체, GPU 최적화.
    * 스크래치 제거: 스크롤 자동 타임라인으로 대체.
    * 오브제: 3D 대신 고화질 WebP 시네마그래프 폴백.
    * 폰트: rem 단위 유동 축소, Bold 두께 유지.

## 6. 개발 검증 및 벤치마크
* 성능 목표: Lighthouse 90점 이상.
* 최적화: WebP/WebM 변환, will-change 하드웨어 가속, 디바운싱 적용.
* CTA 트래킹:
    * button-option-1: [ 완벽한 유리알 장벽 시작하기 ]
    * button-option-2: [ 3일 체험 키트 무상으로 받기 ]
    * button-option-3: [ 내 피부에 맑은 투명도 박제하기 ]
